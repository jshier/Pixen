Sharing a pattern

If you've made a pattern you like, you can send it to others for their use.

Open the saved patterns drawer if it is not already open by clicking on the pattern editor's Saved Patterns toolbar button.

Drag the desired pattern to a Finder window open to the target directory. Pixen will prompt you for a file name.

or

Drag the desired pattern directly from the drawer to Mail, iChat, or one of several other supported Mac applications.

Either option will give you an exported pattern file that others can import and use on their systems.
