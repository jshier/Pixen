Restoring the preview window to the actual size of the document

After working with the preview window at an obscure percentage for a while, you may wish to restore it to its original size. Sure, you could use Size To, but this is faster:

To restore the preview window to the actual size of the document:

Click the action gear in the preview window.

Click Size to Actual

Note that Pixen doesn't always size the window to 100% when using this command: that could have cataclysmic results when working with large images. Instead, it limits the size of the window to 256x256.
