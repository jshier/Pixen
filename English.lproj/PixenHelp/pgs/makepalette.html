Making a custom palette

If you're doing a lot of images for the same project, you probably want to keep a consistent color scheme throughout. Or maybe you have a number of common colors you want to use frequently. Just like an artist's palette in reality, Pixen can help you get the colors you need quickly.

To create a blank custom palette:

  • Open a Pixen palette panel by clicking the Palette icon in a document's toolbar or by clickin the Pixen Colors icon in a color panel's toolbar.
  • Choose New Palette from the action (gear) menu.

To create a new palette based on another one:

  • Open a Pixen palette panel by clicking the Palette icon in a document's toolbar or by clickin the Pixen Colors icon in a color panel's toolbar.
  • Choose the palette on which you want to base your new one.
  • Choose Duplicate Palette from the action (gear) menu.

Once you've got your new palette, click the add button to add new colors to it or double-click a color to change it. You can reorder the colors by dragging them around.
