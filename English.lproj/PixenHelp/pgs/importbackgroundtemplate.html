Importing a background template

If someone has sent you a background template file, you can import it into Pixen for use in your documents.

To import a background template:

Double click the background template file in Finder.

or

Drag the file to Pixen's dock icon.

After doing this, the pattern will appear in the background configurator's user templates area.
