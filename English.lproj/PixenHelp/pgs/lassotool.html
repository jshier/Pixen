Using the lasso tool

The lasso tool allows you to define regions for selection that are too irregular to be grabbed with the rectangular selection or magic wand tools. The lasso tool works like the pencil tool while you're drawing: it fills in pixels you draw over. Once you let go of the mouse button, every pixel enclosed in the region you've outlined will become selected.
