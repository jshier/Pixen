What's New in Pixen?

For this version of Pixen, we've redesigned a number of features and added several more. Now you can create pixel art even more efficiently and effectively. This page provides a brief overview of what's new; to learn more, read through the complete documentation or visit Pixen's web site.

Speed. And more speed.

Yes, Pixen is designed first and foremost for pixel art, but it can be useful for simple graphics editing as well. So now Pixen is speedy even working with extremely large images with multiple layers.

Get animated.

Pixen now allows you to create animations just as easily as you would single images. It's just drag and drop, then you've got an animated GIF or a Quicktime movie on your hands.

Making an animation

A powerful palette.

Pixen can now be used effectively to edit documents with restricted palettes, such as sprites for cell phone games or the like. Additionally, the palette's interface is more intuitive and powerful than ever.

Working with palettes

Infinite tiling.

Pixen's tiling view now can draw infinite tilees, filling the window. Additionally, the drawing is no longer delayed.

Using the tile view

Advanced backgrounds.

The backgrounds system has been redesigned from the ground up to be more flexible, intuitive, and useful to every artist. And the templates can be shared with others

Working with backgrounds

Patterns simplified.

Defining a pattern for your rools is now even easier. And once you make one, you can save it to use later or share with you coworkers.

Setting a tool's pattern

Take another look.

Pixen's preview window is now effective with exceptionally small and large documents. Furthermore, a new action menu in the corner provides additional control.

Working with the preview window
