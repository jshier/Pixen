Using a saved pattern

If you've saved or imported a pattern, you can easily retrieve it and set it on the current tool.

Open the saved patterns drawer if it is not already open by clicking on the pattern editor's Saved Patterns button.

Double-click the desired pattern in the drawer.

or

Select the desired pattern in the drawer.

Click the Load toolbar button in the pattern editor.

The selected pattern will be set as the current tool's pattern, ready to be used in your image.
