Sharing a palette

If you've made a palette you like, you might want to share it with your friends or use it on another computer. Pixen can help.

To export a palette:

  • Open a Pixen palette panel by clicking the Palette icon in a document's toolbar or by clicking the Pixen Colors icon in a color panel's toolbar.
  • Choose the palette you want to export.
  • Choose Export Palette... from the action (gear) menu.

To install a saved palette:

Use the Palette > Install Palette... menu item.

Pixen supports the following palette formats:

  • Pixen Palette (.pxpalette)
  • Microsoft RAW Palette (.pal)
  • Adobe Color Table (.act)
  • Jasc Paint Shop Pro Palette (.pal)
