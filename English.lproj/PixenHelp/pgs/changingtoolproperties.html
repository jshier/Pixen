Changing tool properties

While every tool has been designed to work as you might expect by default, you can customize many tools' behavior in their tool properties window. The pencil tool's properties panel looks like this:

As you can see, this window allows you to set the pencil's line thickness as well as its pattern. You can also see that the title includes "(Left)": the left and right tools have independent preferences, which means that you can have the same tool work in different ways on each mouse button.

To change a tool's properties:

If the tool properties panel is already open, just edit the fields inside, and they'll apply next time you use the tool.

If the panel is closed, you can open it by double-clicking the tool on which you'd like to set properties.
