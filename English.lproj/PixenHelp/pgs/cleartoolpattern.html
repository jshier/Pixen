Clearing a tool's pattern

If you decide that you don't want to use a pattern anymore or want to use a tool size instead, you can clear the pattern and return the tool ot normal.

Select the tool for which you want to clear the pattern

Make sure the Tool Properties panel is open. If it is not, double click the tool's button to open it.

Click the Clear Pattern button.

After doing this, the selected tool will behave as it did before the pattern was set.
